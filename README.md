## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- Jupyter Notebook

### Run Notebooks

```bash
# Clone the repository
git clone https://github.com/Kushwah1/DeepLearning-_Ashutosh_2025-2026.git
cd DeepLearning-_Ashutosh_2025-2026

# Install dependencies
pip install tensorflow numpy pandas matplotlib seaborn jupyter

# Launch Jupyter
jupyter notebook
```
